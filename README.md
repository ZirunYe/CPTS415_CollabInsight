# CPTS415_CollabInsight

An Academic Collaboration Analyzer based on NoSQL
database designed to uncover meaningful collaboration patterns among scientists
and researchers.

Authors: Tony Cao, Julia Lee, Son Nguyen, Zirun Ye

Dataset:
original JSON dataset: https://www.kaggle.com/datasets/mathurinache/citation-network-dataset

reduced  CSV dataset used in early stages: https://www.kaggle.com/datasets/devintheai/citation-networks

reduced CSV dataset created using data_prepare.py: https://kaggle.com/datasets/88c2756017656d379049aad3a6ad5749f8fb64dd51b5291ffe01141f00b9b631
